# lib/

Place external `.jar` libraries here.

This project currently uses only the **Java Standard Library** (`java.util`) — no external JARs required.

| Library Used | Package | Purpose |
|---|---|---|
| Scanner | `java.util.Scanner` | Console input handling |
| ArrayList | `java.util.ArrayList` | Patient queue storage |
| List | `java.util.List` | Interface for patient list |
